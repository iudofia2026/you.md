# Personal Information

This directory contains information about you as a person.

## Files to Create

### Core Files
- **values.md** - Core values and principles
- **traits.md** - Personality traits, work preferences
- **contact.md** - Contact information

### Optional Files
- **schedule.md** - Typical daily schedule
- **fitness.md** - Fitness routines
- **languages.md** - Language proficiency
- **reading.md** - Reading habits
- **recreation.md** - Hobbies and interests

### Goals
Track goals however you prefer:
- Single `goals.md` file
- `annual-goals/` subdirectory with yearly files
- `current-focus.md` for what you're working on now
