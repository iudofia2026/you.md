# Other - Custom Tracking

This directory is for anything that doesn't fit in the organized categories. Your personal context is flexible - add whatever you need.

## Common Custom Files

Feel free to create files for:

| File | Purpose |
|------|---------|
| `habits.md` | Track habits, routines, goals |
| `health.md` | Health metrics, workouts, medications |
| `reading.md` | Books read, reading list, notes |
| `finances.md` | Financial goals, accounts (be careful with privacy!) |
| `ideas.md` | Ideas you want to capture and develop |
| `quotes.md` | Quotes that inspire you |
| `travel.md` | Places visited, travel plans |
| `journal.md` | Personal journal entries |
| `recipes.md` | Favorite recipes, meal plans |

## Creating a Custom File

Just create a markdown file:

```markdown
# My Custom File

**Created**: 2026-03-02
**Purpose**: What this file tracks

## Content
```

No template required - make it work for you.
